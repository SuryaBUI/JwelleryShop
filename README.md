# 💎 Jewellery Shop Management System

A Spring Boot-based web application designed to simplify and manage daily operations of a jewellery shop. The system provides efficient modules for handling new items, old items, and banki services with accurate calculations and easy record management.

## 🚀 Features
🆕 New Item Module
Calculate price of new jewellery items
Input details like weight, rate, and making charges
Save finalized entries
Automatic redirection to homepage after saving
🔄 Old Item Module
Evaluate and calculate value of old jewellery
Supports accurate pricing logic
Save transaction records for future reference
🏦 Banki Services
Manage all banki-related services
Maintain structured records
Track transactions efficiently
## 🛠️ Tech Stack
Backend: Spring Boot
Frontend: HTML, CSS, JavaScript
Database: MySQL
Build Tool: Maven / Gradle (based on your project)
## ⚙️ Prerequisites

Before running the application, make sure you have:

✅ Java 21 installed
✅ MySQL installed and running
✅ Proper database schema created and configured in the application
## ▶️ How to Run

Clone the repository:

1.git clone your-repo-url

2.Open the project in your IDE (IntelliJ / Eclipse / VS Code)

3.Configure database credentials in:

application.properties
4.Run the Spring Boot application

Open your browser and go to:

http://localhost:8080/
